Tema 1 - Texto

Propiedad Valor Inicial Descripción
color inherit Define el color del texto.
font-family sans-serif Especifica la fuente del texto.
font-size medium Define el tamaño del texto.
font-weight normal Define el grosor del texto.
font-style normal Especifica el estilo de la fuente (normal, italic, oblique).
font-variant normal Especifica el estilo de la fuente (normal, small-caps).
font inherit Propiedad abreviada para font-style, font-variant, font-weight, font-size, line-height y font-family.
text-align inherit Especifica la alineación del texto.
line-height normal Especifica la altura de la línea.
text-decoration none Especifica la decoración del texto (subrayado, tachado, etc.).
text-transform none Especifica la transformación del texto (mayúsculas, minúsculas).
vertical-align baseline Especifica la alineación vertical del texto.
text-indent 0 Especifica la sangría del texto.
letter-spacing normal Especifica el espaciado entre letras.
word-spacing normal Especifica el espaciado entre palabras.
white-space normal Especifica cómo se manejan los espacios en blanco.

Tema 2 - Enlaces

Características Propiedades Descripción
Tamaño, Color y Decoración color, font-size, text-decoration Define el tamaño y color de los enlaces.
Decoración Personalizada border, background, padding Especifica la decoración visual de los enlaces.

Tema 3 - Imágenes

Características Propiedades Descripción
Ancho y Altura width, height Define el ancho y la altura de las imágenes.
Eliminar el Borde border: none; Elimina cualquier borde alrededor de la imagen.

Tema 4 - Listas

Propiedad Valor Inicial Descripción
list-style-type disc Especifica el tipo de marcador para listas no ordenadas.
list-style-position outside Especifica la posición del marcador en la lista.
list-style-image none Especifica una imagen como marcador para la lista.
list-style disc Propiedad abreviada para las propiedades de la lista.

Tema 5 - Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Especifica si se deben colapsar los bordes de la tabla.
border-spacing table 0 Especifica el espacio entre los bordes de las celdas.
empty-cells table show Especifica si se deben mostrar las celdas vacías.
caption-side table top Especifica la posición del título de la tabla.
